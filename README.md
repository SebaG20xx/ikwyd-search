# ikwyd-search
Scraper "Quick & Dirty" para revisar si una ip estuvo involucrada en descargas p2p mediante iknowwhatyoudownload.com

# Requerimientos:
- Python 3.3 o superior
- Conexión a Internet
- html5lib
- requests
- BeautifulSoup


# Instrucciones:
1. Insertar tus ips en ips.txt **(1 ip por línea)**
2. Ejecutar
